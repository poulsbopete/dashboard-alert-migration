# Support

## Getting Help

- Use GitHub issues for reproducible bugs and documentation issues.
- Use feature requests for new migration capabilities or repo improvements.
- Use `SECURITY.md` for vulnerability reporting.
- Canonical commands: `docs/command-contract.md`

## Include This Context

- source type: Grafana or Datadog
- exact command used
- sanitized input or minimal fixture
- warnings or errors from the tool
- Kibana / Elasticsearch version if relevant
