# Code of Conduct

We follow Elastic’s community code of conduct for this public repository:
[https://www.elastic.co/community/codeofconduct](https://www.elastic.co/community/codeofconduct).

## Reporting

Email [community@elastic.co](mailto:community@elastic.co) about conduct concerns in
this community. For employee-related misconduct that belongs under corporate
ethics and compliance, use the
[Ethics and Compliance Hotline](https://secure.ethicspoint.com/domain/media/en/gui/74447/index.html).
